## 修改说明

本项目对原始代码进行了以下修改：

### 新增功能
- Android 原生 UI（基于 Jetpack Compose）
- VPN Service 实现本地代理
- 分享链接解析功能（各平台独立 Downloader）
- Android 下载管理系统
- GitHub Releases 自动更新机制
- 多语言支持（中文/英文）
- **动态规则系统**（规划中）- 远程配置 + 声明式规则引擎

### 架构调整
- 采用分享链接解析 + 代理抓取双模式
- 为每个平台实现独立的 Downloader
- 使用 Kotlin 重写了业务逻辑层
- 实现了 Android 特有的生命周期管理
- 远程配置支持热更新平台解析规则

### 技术架构

```
├── 动态规则引擎
│   ├── 远程配置管理
│   ├── 声明式规则引擎
│   └── 配置签名验证
├── 平台适配层
│   ├── 各平台独立 Downloader
│   ├── 平台检测和路由
│   └── API 响应提取
└── 核心功能
    ├── VPN 代理服务
    ├── 下载管理
    └── 资源识别
```

### 与桌面版的差异

| 对比项 | 桌面版 | Android 版（本项目） |
|--------|--------|---------------------|
| 平台支持 | 自动支持所有平台 | 仅支持已实现的平台（支持远程配置热更新） |
| 操作方式 | 在目标 APP 操作即可自动抓取 | 复制分享链接 或 开启代理 |
| 技术实现 | MITM 代理 + 插件系统 | VPN Service + API 解析 + 远程配置 |
| 规则更新 | 随程序更新 | 支持远程配置热更新（规划中） |

**说明**：Android 版针对移动端特点进行了优化，采用分享链接解析作为主要方式，代理抓取作为补充。动态规则系统规划中，将支持远程配置热更新。

## 长期规划

本项目的长期规划包括：

1. **完善动态规则系统**
   - 实现远程配置更新机制

2. **自动化同步**

3. **持续优化**
   - 性能优化
   - 用户体验改进
   - 更多平台支持
