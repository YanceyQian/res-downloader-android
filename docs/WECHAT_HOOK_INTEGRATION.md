## 一、方案概述

针对微信视频号等需要深度解析的场景，提供一种可选增强方案。该方案采用动态分析技术，在不修改系统环境的前提下，提升视频号资源的获取成功率。

### 1.1 基本原理

```
应用进程
 ┌─────────────────────────────┐
 │  目标应用                    │
 │  ┌───────────────────────┐  │
 │  │  动态分析模块          │  │
 │  │  (嵌入式组件)          │  │
 │  └───────────┬───────────┘  │
 │              │               │
 │              ▼               │
 │  ┌───────────────────────┐  │
 │  │  数据采集             │  │
 │  │  → 视频地址           │  │
 │  │  → 解密参数           │  │
 │  └───────────────────────┘  │
 └──────────────┬──────────────┘
                │
                ▼
 ┌─────────────────────────────┐
 │  主应用                      │
 │  接收 → 解密 → 保存          │
 └─────────────────────────────┘
```

### 1.2 优缺点对比

| 项目 | 说明 |
|------|------|
| ✅ 无需 Root | 直接安装即可使用 |
| ✅ 成功率较高 | 直接获取数据源，不依赖网络 API |
| ✅ 不受证书限制 | 在进程内部收集数据 |
| ⚠️ 需要用户确认风险 | 使用前需阅读并同意免责条款 |
| ⚠️ 依赖目标应用版本 | 目标应用更新后可能需要同步适配 |

## 二、方案说明

### 2.1 默认获取方式（无需增强方案）

以下方式可直接使用，无需额外配置：

| 方式 | 说明 | 操作 |
|------|------|------|
| **分享链接解析** | 复制视频分享链接到应用 | 复制 → 粘贴 → 下载 |
| **VPN 代理抓取** | 开启 VPN 后自动捕获流量 | 开启代理 → 浏览内容 → 下载 |

### 2.2 增强方案概览

增强方案适用于默认方式无法获取资源的场景，例如：

- 朋友圈内嵌视频
- 小程序内部视频
- 其他深层嵌套内容

> **注意**：增强方案为可选功能，不启用不影响其他功能使用。
