# 爱享素材下载器 - Android 用户使用指南

本文档为 Res-Downloader Android 应用的用户提供详细的使用说明。

## 应用简介

爱享素材下载器是一款简洁易用的网络资源下载工具，支持分享链接解析和代理抓取两种方式获取视频、音频、图片等资源。本项目是 [putyy/res-downloader](https://github.com/putyy/res-downloader) 的 Android 移植版本。

### ⚠️ 与桌面版的差异

本应用是针对 Android 平台特点重新设计的，**工作方式与桌面版不同**：

| 对比项 | 桌面版 | Android 版（本应用） |
|--------|--------|---------------------|
| 下载方式 | 代理拦截（透明抓取） | 分享链接解析 + VPN 代理抓取 |
| 平台支持 | 自动支持所有平台 | 仅支持已实现的平台 |
| 操作方式 | 在目标 APP 操作即可自动抓取 | 复制分享链接 或 开启代理抓取 |
| HTTPS 拦截 | 需要安装证书/代理转发 | VPN Service 可直接拦截 |

> **详细说明**：Android 版通过 VPN Service 实现代理抓取，可以拦截所有 HTTPS 流量。但与桌面版不同，Android 版无法自动识别和抓取所有平台资源。需要为每个平台单独实现 API 解析逻辑。目前已支持：抖音、快手、小红书、B站、网易云音乐、微信视频号等；代理抓取还支持：QQ音乐、酷狗音乐（仅代理抓取）。

### 主要功能

- 🔗 **分享链接解析**：复制分享链接直接解析下载（推荐）
- 🔒 **VPN 代理抓取**：开启代理后自动捕获应用中的资源
- 📱 **多平台支持**：抖音、快手、小红书、B站、网易云音乐、微信视频号、QQ音乐、酷狗音乐
- 🎬 **M3U8 支持**：自动解析并下载 m3u8 流媒体视频
- 📥 **下载管理**：支持下载队列、进度显示、断点续传
- 🔄 **自动更新**：集成 GitHub Releases 自动检测和安装更新
- 🌐 **多语言**：支持简体中文和英文

主界面分为三个标签页：

1. **主页**：代理控制和资源列表

主页标签显示已拦截的资源列表，您可以：

下载标签管理下载任务：
- **下载中**：显示正在下载的任务和进度

#### 设置标签

设置标签提供各种配置选项：
- 基础设置（语言、下载目录、清晰度）
- 文件命名规则
- 网络设置（代理端口、上游代理等）
- 高级设置（证书安装、电池优化等）
- 关于应用

## 使用步骤

### 方式一：分享链接解析（推荐）

这是最简单直接的方式，适用于已支持的平台：

1. 在目标应用中复制分享链接
2. 打开爱享素材下载器
3. 链接会自动粘贴到输入框
4. 点击「解析」按钮
5. 选择资源后点击「下载」

**支持的分享链接类型**：
- 抖音视频分享链接
- 快手视频分享链接
- 小红书笔记分享链接
- B站视频分享链接
- 网易云音乐分享链接
- 微信视频号分享链接

### 方式二：代理抓取

如果分享链接解析不支持某个资源，可以使用代理抓取方式：

#### 第一步：启动代理

1. 打开应用，进入「主页」标签
2. 点击代理开关，开启代理服务
3. 首次使用需要授予 VPN 权限
4. 代理启动后，开关显示绿色

> **注意**：代理启动后，手机的网络流量会通过本地代理，这可能会略微影响网速。

### 第二步：安装 CA 证书（推荐）

为了能够拦截 HTTPS 流量，建议安装 CA 证书：

1. 进入「设置」标签
2. 点击「安装证书」
3. 按照提示下载证书压缩包
4. 解压后得到 ca.crt 证书文件
5. 打开手机「设置」→「安全」→「加密与凭据」
6. 选择「从存储设备安装证书」
7. 选择证书文件，命名后确认安装

> **安全提示**：安装证书后会增加安全风险，请仅在使用下载功能时启用代理，完成后及时关闭。

### 第三步：访问资源

在外部浏览器或应用中访问你想下载的资源，例如：
- 打开抖音，播放一个视频
- 打开微信，播放视频号内容
- 打开小红书，查看图文或视频
- 打开酷狗音乐，播放一首歌曲

### 第四步：下载资源

返回爱享素材下载器应用：
1. 在主页的资源列表中找到需要下载的内容
2. 点击对应资源项的「下载」按钮
3. 进入「下载」标签查看下载进度
4. 下载完成后，点击「打开」查看或「分享」发送

## 常见问题

### Q1：分享链接解析和代理抓取有什么区别？

**分享链接解析**（推荐）：
- 只需复制分享链接，粘贴到应用即可
- 简单快捷，无需开启代理
- 支持所有已实现分享链接解析的平台

**代理抓取**：
- 需要开启代理服务
- 可以在应用中浏览并自动捕获资源
- 适用于分享链接不支持的资源

### Q2：为什么推荐分享链接解析？

分享链接解析方式更简单：
1. 无需开启代理，节省电量
2. 不需要安装证书
3. 不影响其他应用的网络使用
4. 解析速度更快

### Q3：为什么有些资源下载失败？

可能原因：
1. **资源已失效**：分享链接可能已过期
2. **平台限制**：部分平台有防下载机制
3. **网络问题**：网络不稳定导致下载中断
4. **未支持平台**：请尝试代理抓取方式

**解决方法**：
1. 检查资源链接是否有效
2. 切换到稳定的网络环境
3. 尝试使用代理抓取方式

### Q4：拦截到太多不相关的资源怎么办？

**方法一**：在设置中关闭「全量拦截」，只拦截指定平台

**方法二**：使用「域名规则」进行精细过滤，在设置 → 域名规则中编辑：
```
*                 # 匹配所有
*.douyin.com       # 匹配抖音
*.kuaishou.com     # 匹配快手
```

### Q5：某某网址/应用无法下载？

Android 版与桌面版不同，需要为每个平台单独实现支持：

**解决方法**：
- 使用分享链接解析方式（推荐）
- 尝试使用代理抓取方式
- 等待后续版本支持

### Q6：无法正常获取资源？

1. 检查网络连接是否正常
2. 确认平台是否在支持列表中
3. 尝试使用另一种方式（分享链接解析 或 代理抓取）
4. 检查应用是否有网络权限

### Q7：关闭软件后无法正常上网？

正常情况下关闭代理后即可恢复上网。如果仍有问题：
1. 确认代理已完全关闭（检查通知栏）
2. 重启手机网络（开启/关闭飞行模式）
3. 重启应用

### Q8：提示「链接不是私密连接」？

通常是证书未正确安装：
1. 在设置中重新下载并安装证书
2. 确保证书安装到「系统」级别（部分手机需要）
3. 重启应用后重试

### Q8：拦截不到小程序中的资源？

尝试清理微信缓存：
1. 微信设置 → 存储空间 → 清理缓存
2. 删除小程序后重新打开
3. 重新浏览内容

### Q9：如何只拦截当前观看的视频号视频？

关闭「全量拦截」，打开视频号视频详情页（通常是分享给好友后打开的页面），这样通常只会拦截到当前播放的内容。

### Q10：如何拦截某个视频号账号的所有视频？

直接打开该作者的个人主页，逐个浏览视频即可。

### Q11：下载速度慢、大视频下载失败？

**建议解决方案**：
1. 使用更稳定的 WiFi 网络
2. 在设置中调整「同时下载任务数」
3. 大文件建议在充电时下载，避免系统省电策略中断
4. 可以复制链接后使用其他下载工具（如 ADM 等）

### Q12：M3U8 视频如何预览和下载？

本应用支持自动识别和下载 M3U8 视频。如果需要预览：
- 可以先下载后再播放
- 或复制 M3U8 链接到支持的播放器中预览

### Q13：如何安装 CA 证书？

请参考上文「第二步：安装 CA 证书」的详细步骤。

### Q14：代理开启后无法上网？

1. 确认代理是否正常运行（主页显示绿色开关）
2. 重启代理：关闭后再开启
3. 重启应用
4. 检查是否授予了 VPN 权限

### Q15：如何关闭代理？

方法一：在应用主页关闭代理开关
方法二：点击通知栏中的「停止代理」按钮
方法三：关闭应用（会自动停止代理）

### Q16：下载的文件保存在哪里？

默认保存位置：`/storage/emulated/0/Download/ResDownloader/`

您可以在设置中修改下载目录，选择任意您喜欢的文件夹。

### Q17：如何更新应用？

1. 进入「设置」标签
2. 点击「检查更新」
3. 如果有新版本，点击「立即更新」
4. 下载完成后自动弹出安装界面
5. 按提示完成安装

### Q18：后台下载时应用被杀死了？

部分手机系统会限制后台运行，需要进行以下设置：

**华为/荣耀**：
- 设置 → 应用 → 爱享素材下载器 → 电池 → 启动管理 → 允许后台运行
- 设置 → 应用 → 爱享素材下载器 → 电池 → 耗电防火墙 → 关闭

**小米/红米**：
- 设置 → 应用设置 → 应用管理 → 爱享素材下载器 → 电池 → 无限制
- 设置 → 电池与性能 → 应用省电策略 → 爱享素材下载器 → 无限制

**OPPO/vivo**：
- 设置 → 电池 → 耗电保护 → 爱享素材下载器 → 关闭后台冻结
- 设置 → 电池 → 后台耗电管理 → 爱享素材下载器 → 允许后台运行

**通用**：
- 设置 → 电池 → 电池优化 → 爱享素材下载器 → 不优化

### Q19：为什么下载速度慢？

1. **网络问题**：尝试切换 WiFi 或移动数据
2. **平台限速**：部分平台对单个 IP 限速
3. **服务器负载**：高峰时段下载较慢
4. **代理影响**：代理会增加延迟

## 注意事项

### 安全提示

1. 🔒 本应用仅供用户下载自己上传到各大平台的网络资源
2. 🔒 请遵守各平台的服务条款和版权政策
3. 🔒 安装 CA 证书会降低设备安全性，仅在需要时启用
4. 🔒 定期更新应用以获得最新安全修复

### 隐私提示

1. 📱 应用需要网络权限来拦截和下载资源
2. 📱 所有代理流量都在本地处理，不会上传到服务器
3. 📱 CA 证书仅用于解密 HTTPS 流量，不会收集个人信息

### 使用建议

1. ✅ 下载完成后及时关闭代理，节省电量和流量
2. ✅ 定期清理下载列表，释放存储空间
3. ✅ 建议使用稳定的 WiFi 环境下载大文件
4. ✅ 遇到问题先查看常见问题解答，大部分问题可以自行解决
