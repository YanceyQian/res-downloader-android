# 用户使用指南

本文档为爱享素材下载器 Android 应用的用户提供详细的使用说明。

## 应用简介

爱享素材下载器是一款基于代理抓包技术的网络资源下载工具，支持从多个平台下载视频、音频、图片等资源。

### 主要功能

- 🔒 **代理抓包**：通过本地代理拦截网络请求
- 📱 **多平台支持**：微信视频号、抖音、快手、小红书等
- 🎵 **音频下载**：支持酷狗音乐、QQ 音乐等
- 🎬 **视频下载**：支持普通视频和 M3U8 流媒体
- 🔄 **自动更新**：自动检测并安装应用更新

## 使用步骤

### 第一步：启动代理

1. 打开应用，进入"主页"标签
2. 点击代理开关，开启代理服务
3. 首次使用需要：
   - 允许 VPN 权限
   - 允许通知权限（用于显示状态）
4. 代理启动后，开关显示绿色，状态显示"代理运行中"

**注意**：代理启动后，手机的所有网络流量都会通过本地代理，这可能会略微影响网速。

### 第二步：配置系统代理（可选）

部分应用可能不会自动使用 VPN 代理，需要手动配置：

1. 进入手机设置 → WLAN
2. 长按已连接的 WiFi → 修改网络
3. 高级选项 → 代理 → 手动
4. 代理主机名：`127.0.0.1`
5. 代理端口：`8899`（可在设置中修改）

### 第三步：访问资源

在外部浏览器或应用中访问你想下载的资源，例如：

- 打开抖音，播放一个视频
- 打开微信，播放视频号内容
- 打开小红书，查看图文或视频

### 第四步：下载资源

返回爱享素材下载器应用：

1. 在资源列表中找到需要下载的内容
2. 点击对应资源项的"下载"按钮
3. 进入"下载"标签查看下载进度
4. 下载完成后，点击"打开"查看或"分享"发送

## 常见问题

### Q1：为什么有些资源下载失败？

可能原因：
1. **需要 CA 证书**：HTTPS 流量需要安装 CA 证书才能解密
2. **资源已失效**：分享链接可能已过期
3. **平台限制**：部分平台有防下载机制
4. **网络问题**：网络不稳定导致下载中断

**解决方法**：
1. 在设置中安装 CA 证书
2. 检查资源链接是否有效
3. 切换到稳定的网络环境

### Q2：如何安装 CA 证书？

1. 进入应用"设置"标签
2. 点击"安装证书"
3. 按照提示下载证书文件
4. 打开手机设置 → 安全 → 加密与凭据 → 从存储设备安装
5. 选择下载的证书文件
6. 设置证书名称（任意），确认安装

**注意**：安装证书后会增加安全风险，请仅在使用下载功能时启用代理，完成后及时关闭。

### Q3：代理开启后无法上网？

1. 确认代理是否正常运行（主页显示"代理运行中"）
2. 重启代理：关闭后再开启
3. 重启应用
4. 检查是否正确设置了系统代理

### Q4：如何关闭代理？

方法一：在应用主页关闭代理开关
方法二：点击通知栏中的"停止代理"按钮
方法三：关闭应用（会自动停止代理）

### Q5：下载的文件保存在哪里？

默认保存位置：`/storage/emulated/0/Download/ResDownloader/`

可以在设置中修改下载目录，选择任意你喜欢的文件夹。

### Q6：如何更新应用？

1. 进入"设置"标签
2. 点击"检查更新"
3. 如果有新版本，点击"立即更新"
4. 下载完成后自动弹出安装界面
5. 按提示完成安装

### Q7：后台下载时应用被杀死了？

部分手机系统会限制后台运行，需要进行以下设置：

1. **华为/荣耀**：
   - 设置 → 应用 → 爱享素材下载器 → 电池 → 启动管理 → 允许后台运行
   - 设置 → 应用 → 爱享素材下载器 → 电池 → 耗电防火墙 → 关闭

2. **小米/红米**：
   - 设置 → 应用设置 → 应用管理 → 爱享素材下载器 → 电池 → 无限制
   - 设置 → 电池与性能 → 应用省电策略 → 爱享素材下载器 → 无限制

3. **OPPO/vivo**：
   - 设置 → 电池 → 耗电保护 → 爱享素材下载器 → 关闭后台冻结
   - 设置 → 电池 → 后台耗电管理 → 爱享素材下载器 → 允许后台运行

4. **通用**：
   - 设置 → 电池 → 电池优化 → 爱享素材下载器 → 不优化

### Q8：为什么下载速度慢？

1. **网络问题**：尝试切换 WiFi 或移动数据
2. **平台限速**：部分平台对单个 IP 限速
3. **服务器负载**：高峰时段下载较慢
4. **代理影响**：代理会增加延迟

## 注意事项

### 安全提示

1. 🔒 本应用仅用于下载你有合法使用权的资源
2. 🔒 请遵守各平台的服务条款和版权政策
3. 🔒 安装 CA 证书会降低设备安全性，仅在需要时启用
4. 🔒 定期更新应用以获得最新安全修复

### 隐私提示

1. 📱 应用需要网络权限来拦截和下载资源
2. 📱 所有代理流量都在本地处理，不会上传到服务器
3. 📱 CA 证书仅用于解密 HTTPS 流量，不会收集个人信息

### 使用建议

1. ✅ 下载完成后及时关闭代理，节省电量和流量
2. ✅ 定期清理下载列表，释放存储空间
3. ✅ 建议使用稳定的 WiFi 环境下载大文件
4. ✅ 遇到问题先查看常见问题解答，大部分问题可以自行解决
